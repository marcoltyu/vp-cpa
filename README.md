# VPCPA Static HTML v1

This is the first static website version for Venture Partners CPA Limited.

## Structure

index.html
assets/css/styles.css
assets/js/main.js
assets/images/

## How to preview

Open index.html directly in a browser.

## How to deploy

Upload the full contents of this folder to the web root of the hosting account.

For example, upload these files into public_html or the domain document root.

## Important notes before launch

1. Replace the VP text mark with the official logo if available.
2. The enquiry form currently uses mailto and opens the visitor's email application.
3. For a production contact form, connect the form to Formspree, Netlify Forms, Cloudflare Workers, or the hosting provider's form handler.
4. Add favicon and Open Graph image before formal launch.
5. Add Google Search Console and analytics if required.
